# Phishing-Email-Forensics
Email header analysis with SPF, DKIM, DMARC validation and domain analysis toolkit  


Executive Summary


An in-depth analysis of a suspicious email received through the corporate email gateway. The email was isolated in a sandboxed virtual environment and 
subjected to multi-layered analysis techniques, including header inspection, URL reputation analysis, and threat intelligence gathering. 

Based on the results, it is concluded that the email is a phishing attempt designed to lure users into clicking a malicious link
